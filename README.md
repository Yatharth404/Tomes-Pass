# Tomes Pass

## Introduction

Tomes Pass is a secure, offline password manager built with Electron. It allows you to store your passwords locally on your device, encrypted and protected by a PIN, ensuring your sensitive information remains private and accessible only to you.

## Download

Download the latest version of Tomes Pass from our [GitHub Releases](https://github.com/Yatharth404/Tomes-Pass/releases/latest/download/Tomes.Pass.Setup.exe).

## Website

Visit our website at [tomespass.vercel.app](https://tomespass.vercel.app) for more information and updates.

## Detailed App Info

### Features

- **Offline Storage**: All passwords are stored locally on your device, ensuring no data is sent to external servers.
- **Encryption**: Passwords are encrypted using strong cryptographic methods, with keys derived from your PIN.
- **PIN Protection**: Access to your password vault is secured by a personal identification number (PIN).
- **Simple Interface**: Easy-to-use interface for adding, viewing, and managing your passwords.

### How to Use

1. **Installation**: Download and install the app from the GitHub releases page.
2. **Setup**: On first launch, set up your PIN to secure your vault.
3. **Add Passwords**: Use the interface to add new password entries with titles, usernames, and passwords.
4. **Access Passwords**: Enter your PIN to unlock and view your stored passwords.
5. **Security**: Your data is encrypted and stored in a hidden directory on your system.

### Security

Tomes Pass prioritizes your security:
- Passwords are encrypted before storage.
- The encryption key is protected by your PIN using bcrypt hashing.
- Data is stored in your home directory under `.tomes-pass`, keeping it separate from the app installation.
