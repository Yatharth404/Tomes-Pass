# Security Policy

Tomes Pass is designed to keep your passwords secure by storing data locally on your device and protecting it with strong encryption.

## Security Principles

- All password data is encrypted before it is saved.
- Encryption keys are derived from your PIN.
- Sensitive data is stored in a hidden directory under your home folder.
- No password data is transmitted to external servers.

## Responsible Disclosure

If you discover a security vulnerability in Tomes Pass, please report it immediately.

Contact: jainyatharth@hotmail.com

When reporting a vulnerability, include:

- A clear description of the issue
- Steps to reproduce it
- The environment where it occurs (operating system, app version)

We take security seriously and will respond as quickly as possible.
